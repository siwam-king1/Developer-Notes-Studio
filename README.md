# ✨ Features

* 🧠 Paste AI-generated notes from AI
* 🎨 Beautiful futuristic themes
* ⚡ Real-time markdown rendering
* 💻 Syntax-highlighted code blocks
* 📄 Ultra-HD PDF export
* 🖼 PNG/JPG export support
* 🌈 Full live color customization
* 📱 Fully responsive UI
* 🪄 Glassmorphism + neon effects
* 📚 Built-in help/documentation panel
* 💾 Auto-save with localStorage
* ✂ Manual page breaks for PDFs
* 🖱 Drag & drop support
