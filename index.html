Export Guide

PDF Export

Exports selectable text as A4 PDF with proper margins, page numbers, and headers.

PNG / JPG Export

Renders preview at 2x resolution for crisp retina quality. Ultra HD mode exports at 4x.

HTML Export

Exports a standalone HTML file with all styles embedded.

  • A4 Landscape — Wide format
  • Instagram Post — 1080×1080
  • Mobile Wallpaper — 1080×1920
  • Desktop Wallpaper — 2560×1440
